```
$ pip install -r requirements.txt
```

## Controls
- Esc: Quit Application
- T: Takeoff
- W/S: Fly Forward/Back
- A/D: Pan Left/Right
- Q/E: Fly Up/Down
- Z/C: Fly Left/Right
- P: Capture Image
- L: To Land
